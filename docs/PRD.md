## 1. 产品概述

### 1.1 产品愿景
让每个人都能用专业思维做出更好的决策

### 1.2 产品定位
免费、专业、易用的跨领域决策分析工具

### 1.3 目标用户
- 职场人士：职业选择、跳槽评估
- 投资者：投资决策、风险评估
- 创业者：商业决策、机会评估
- 大学生：专业/职业规划
- 所有面临生活决策的人

### 1.4 核心价值
- **专业性**：CPA、CFA、CMA、法考四大领域知识支撑
- **易用性**：无需专业知识即可使用
- **可视化**：展示完整思维推导过程
- **免费性**：完全免费，无广告

## 3. 用户流程

### 3.1 浏览流程

```
用户访问首页
    ↓
选择知识库或使用搜索
    ↓
浏览/搜索模型列表
    ↓
点击模型进入详情页
    ↓
阅读模型内容
    ↓
（可选）收藏模型
```

### 3.2 搜索流程

```
用户输入关键词
    ↓
系统搜索匹配模型
    ↓
展示搜索结果
    ↓
用户点击结果
    ↓
进入详情页
```

## 5. 数据结构

### 5.1 知识库文件

```markdown
# 文件命名规范
cpa-knowledge.md
cfa-knowledge.md
cma-knowledge.md
law-exam-knowledge.md

# 文件结构
# 一级标题：领域名称
## 二级标题：章节名称
### 三级标题：模型名称
| 表格 | 用于模型对比 |
``` 表格 | 用于快速参考 |
```

### 5.2 模型元数据

```json
{
  "id": "npv",
  "name": "净现值（NPV）",
  "category": "CPA",
  "subcategory": "投资评估",
  "formula": "NPV = Σ [CF_t / (1+r)^t] - 初始投资",
  "tags": ["投资", "长期决策", "财务"],
  "applications": ["创业", "职业选择", "买房"]
}
```
