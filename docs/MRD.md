# MRD：市场需求文档

## 项目名称
Decision Assistant Pro - 跨领域决策分析工具

## 1. 市场机会

### 1.1 市场规模
| 领域 | 目标用户 | 市场估算 |
|------|----------|----------|
| 职业决策 | 职场人群（4亿+） | 千亿级 |
| 投资决策 | 投资人群（2亿+） | 百亿级 |
| 生活决策 | 全体网民（10亿+） | 千亿级 |

### 1.2 市场趋势
- **决策焦虑**已成为现代社会的普遍问题
- 越来越多的年轻人开始注重"理性决策"
- 知识付费市场持续增长（2023年中国知识付费市场规模突破1000亿）
- 轻量级工具类产品更受欢迎

## 2. 目标用户

### 2.1 核心用户画像

#### 用户画像 1：职场纠结者
- **年龄**：25-35岁
- **场景**：职业选择、跳槽评估、薪资谈判
- **痛点**：害怕做出错误选择
- **需求**：系统性分析框架

#### 用户画像 2：投资新手
- **年龄**：28-40岁
- **场景**：股票/基金/数字货币投资
- **痛点**：追涨杀跌、情绪化决策
- **需求**：风险评估工具、心理偏差检测

#### 用户画像 3：创业者
- **年龄**：30-45岁
- **场景**：商业决策、合伙人选择
- **痛点**：信息过载、难以权衡
- **需求**：多维度分析模型

### 2.2 用户需求分析

| 需求类型 | 需求描述 | 优先级 |
|----------|----------|--------|
| 核心需求 | 获得专业的决策分析 | P0 |
| 核心需求 | 理解背后的推理逻辑 | P0 |
| 期望需求 | 快速找到相关模型 | P1 |
| 期望需求 | 生成可分享的报告 | P1 |
| 兴奋需求 | AI对话式交互 | P2 |
| 兴奋需求 | 个性化推荐 | P2 |

## 4. 产品功能需求

### 4.1 功能列表

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| 知识库浏览 | 浏览 CPA/CFA/CMA/法律 四大领域 | P0 |

### 4.2 用户故事

#### 用户故事 1
> 作为一名职场人，我想快速找到"机会成本"的定义和应用场景，以便做出更好的职业选择。

#### 用户故事 2
> 作为一名投资者，我想检测自己的投资决策是否受到"损失厌恶"的影响，避免情绪化交易。

#### 用户故事 3
> 作为一名创业者，我想用"NPV"模型评估一个项目的长期价值，做出科学的投资决策。
