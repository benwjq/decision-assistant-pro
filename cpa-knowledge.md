# CPA 知识库

## 1. 财务成本管理

### 成本分类
| 类型 | 定义 | 举例 |
|------|------|------|
| 显性成本 | 直接发生的货币支出 | 学费、工资、租金 |
| 隐性成本 | 机会成本/潜在损失 | 放弃的收入、时间消耗 |
| 沉没成本 | 已发生无法收回 | 过去投入的时间、金钱 |

### 成本效益分析框架
```
决策净收益 = 预期收益 - 显性成本 - 隐性成本 - 风险调整
```

---

## 2. 财务分析方法

### 杜邦分析法
```
ROE = 净利率 × 资产周转率 × 权益乘数
```

### 折现模型
- **DCF（折现现金流）**: 将未来现金流折现到现值
- **折现率选择**: 无风险利率 + 风险溢价（通常 8-12%）

---

## 3. 预算管理

### 零基预算 vs 增量预算
- **零基预算**: 从零开始，每项支出都需要论证
- **增量预算**: 基于上期调整

## 5. 决策相关准则

### 本量利分析 (CVP)
```
利润 = 收入 - 变动成本 - 固定成本
盈亏平衡点 = 固定成本 / (单价 - 单位变动成本)
```

### 边际贡献
```
边际贡献 = 单价 - 单位变动成本
```
