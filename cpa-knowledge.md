# CPA 知识库

## 1. 财务成本管理

### 成本分类
| 类型 | 定义 | 举例 |
|------|------|------|
| 显性成本 | 直接发生的货币支出 | 学费、工资、租金 |
| 隐性成本 | 机会成本/潜在损失 | 放弃的收入、时间消耗 |
| 沉没成本 | 已发生无法收回 | 过去投入的时间、金钱 |

### 成本效益分析框架
```
决策净收益 = 预期收益 - 显性成本 - 隐性成本 - 风险调整
```

---

## 2. 财务分析方法

### 杜邦分析法
```
ROE = 净利率 × 资产周转率 × 权益乘数
```

### 折现模型
- **DCF（折现现金流）**: 将未来现金流折现到现值
- **折现率选择**: 无风险利率 + 风险溢价（通常 8-12%）

---

## 3. 预算管理

### 零基预算 vs 增量预算
- **零基预算**: 从零开始，每项支出都需要论证
- **增量预算**: 基于上期调整

## 5. 决策相关准则

### 本量利分析 (CVP)
```
利润 = 收入 - 变动成本 - 固定成本
盈亏平衡点 = 固定成本 / (单价 - 单位变动成本)
```

### 边际贡献
```
边际贡献 = 单价 - 单位变动成本
```

## 6. CPA 决策模型（人生应用版）

| CPA财务模型 | 本质问题 | 可应用的人生场景 | 现实决策提问（可直接自问） |
|-------------|----------|------------------|--------------------------|
| **1 货币时间价值（TVM）** | 时间是否被正确定价 | 读研 / 转行 / 创业 | 今天损失3年收入，未来能否指数增长？ |
| **2 净现值（NPV）** | 长期是否真正赚钱 | 婚姻 / 职业路径 | 折现后，这条路是正收益还是负收益？ |
| **3 内部收益率（IRR）** | 成长速度 | 技能学习 | 这个能力每年让我提升多少？ |
| **4 投资回收期** | 风险暴露时间 | 创业 / 出国 | 最坏情况下多久能回本？ |
| **5 风险收益权衡** | 风险是否值得 | 高薪跳槽 | 收入增加是否匹配不确定性？ |
| **6 CAPM风险补偿** | 承担风险是否得到溢价 | 自由职业 | 我承担的波动有没有回报？ |
| **7 敏感性分析** | 哪个变量最危险 | 买房 / 创业 | 哪一个变化会让我崩盘？ |
| **8 情景分析** | 多未来推演 | 城市选择 | 3种未来里，我都能活下去吗？ |
| **9 机会成本** | 放弃了什么 | 结婚 / 留职 | 我真正放弃的是哪条人生？ |
| **10 沉没成本** | 是否被过去绑架 | 感情 / 职业 | 如果今天重新开始，我还选吗？ |
| **11 边际分析** | 再投入是否值得 | 再考证 / 再坚持 | 多一年投入带来多少新增收益？ |
| **12 相关成本原则** | 只看未来 | 转型 | 过去努力是否与未来有关？ |
| **13 本量利分析（CVP）** | 模式是否可持续 | 副业 / 内容创作 | 做到多少规模才赚钱？ |
| **14 盈亏平衡点** | 生存底线 | 自由职业 | 每月最低收入是多少？ |
| **15 杠杆效应** | 小变化是否放大结果 | 创业 / 投资 | 收入下降10%是否致命？ |
| **16 WACC资本成本** | 最低可接受回报 | 是否离职 | 新选择是否超过当前稳定收益？ |
| **17 EVA经济增加值** | 是否真正创造价值 | 工作意义 | 忙碌是否超过资源消耗？ |
| **18 现金流优先** | 能否活下去 | 创业初期 | 有没有稳定现金流？ |
| **19 SWOT分析** | 位置判断 | 职业定位 | 我是在优势区竞争吗？ |
| **20 竞争战略（差异化）** | 是否在红海 | 个人IP | 我为什么不可替代？ |
| **21 内部控制思维** | 如何避免灾难 | 婚姻 / 财务 | 是否有风险隔离机制？ |
| **22 风险分散模型** | 不把命运押一处 | 收入结构 | 是否只有单一收入来源？ |
