## 1. 战略管理

### 波特五力模型
| 维度 | 分析内容 |
|------|----------|
| 供应商议价能力 | 原材料供应集中度、价格敏感度 |
| 买方议价能力 | 客户集中度、价格敏感度 |
| 新进入者威胁 | 壁垒高低、规模经济 |

| 现有竞争者 | 行业增长率、产品差异化 |

```
S (优势) W (劣势) + O (机会) T (威胁)

```

### PESTEL 分析
- P: 政治

- S: 社会
- T: 技术
- E: 环境

## 4. 决策分析

### 相关成本分析
```
增量成本 = 差量成本 = 边际成本
```

### 特殊订单决策
```
是否接受特殊订单？
= (特殊单价 - 单位变动成本) × 数量 > 专属成本?
```

### 自制 vs 外购
```
自制成本 = 直接材料 + 直接人工 + 变动制造费用 + 专属固定成本
外购成本 = 采购价格 + 节省的变动成本
→ 选择成本较低者
```

### 关闭 vs 继续
```
继续贡献边际 > 关闭后固定成本？→ 继续
否则 → 关闭
```

## 5. 预算管理

### 全面预算构成
```
销售预算 → 生产预算 → 直接材料/人工/制造费用预算
→ 完工产品成本预算 → 销售及管理费用预算
→ 预计利润表/资产负债表/现金流量表
```

### 预算编制方法
| 方法 | 特点 |
|------|------|
| 零基预算 | 每期从零开始 |
| 滚动预算 | 持续更新（12个月滚动） |
| 弹性预算 | 按不同业务量编制 |
| 作业预算 | 基于作业成本法 |

## 6. 差异分析

### 销售差异
```
销售量差异 = (实际销量 - 预算销量) × 预算单价 × 预算贡献边际率
销售价格差异 = (实际单价 - 预算单价) × 实际销量
```

### 成本差异
```
直接材料差异 = (实际用量 - 标准用量) × 标准价格
直接人工效率差异 = (实际工时 - 标准工时) × 标准工资率
```

## 7. 投资评估

### NPV 净现值
```
NPV = Σ [CF_t / (1+r)^t] - 初始投资
> 0 → 接受
< 0 → 拒绝
```

### IRR 内部收益率
使 NPV = 0 的折现率
> 资本成本 → 接受

### 回收期
| 类型 | 计算 |
|------|------|
| 静态回收期 | 累计现金流回本时间 |
| 动态回收期 | 折现后累计现金流回本时间 |

## 9. CMA 思维模型（人生应用版）

| CMA 思维模型 | 管理会计中的含义 | 对应人生场景 | 可直接自问的问题 |
|--------------|------------------|--------------|------------------|
| **1 成本性态（固定/变动成本）** | 成本结构决定风险 | 买房 / 自由职业 | 我的生活固定支出是否过高？ |
| **2 边际贡献（Contribution Margin）** | 新增收入减新增成本 | 副业 / 社交 | 这件事真正留下多少净收益？ |
| **3 盈亏平衡点** | 最低生存规模 | 转型 / 创业 | 我每月最低需要多少才能安全？ |
| **4 机会成本** | 放弃的最佳替代方案 | 职业选择 | 我放弃的真正是什么？ |
| **5 相关成本原则** | 只考虑未来影响 | 换行业 | 过去投入是否应该忽略？ |
| **6 沉没成本识别** | 历史投入不可追回 | 感情 / 职业 | 如果重新开始，我还会选吗？ |
| **7 约束理论（瓶颈思维）** | 系统由最慢环节决定 | 成长停滞 | 真正限制我的是哪一项能力？ |
| **8 偏差分析（Variance Analysis）** | 计划与实际差异 | 目标管理 | 失败是执行问题还是假设错误？ |
| **9 持续改进（Kaizen）** | 小幅长期优化 | 自我成长 | 是否每天微小进步？ |
| **10 作业成本法（ABC）** | 成本来自具体活动 | 精力管理 | 哪些行为最消耗我？ |
| **11 多维绩效指标** | 不只看利润 | 人生平衡 | 我是否只用收入评价人生？ |
| **12 内部控制思维** | 预防错误机制 | 财务/关系 | 是否有防冲动机制？ |
| **13 资本预算（NPV/IRR）** | 长期投资决策 | 读研 / 创业 | 长期回报是否覆盖投入？ |
| **14 现金流优先** | 生存重于利润 | 职业转型 | 是否有稳定现金来源？ |
| **15 风险—收益权衡** | 回报来自风险承担 | 跳槽 | 风险是否被合理补偿？ |
| **16 敏感性分析** | 关键变量变化测试 | 买房 / 创业 | 哪个变化会让我失败？ |
| **17 情景规划** | 多未来路径 | 城市选择 | 最坏情况我能接受吗？ |
| **18 杠杆效应** | 放大收益与风险 | 内容/IP | 哪件事具有指数放大力？ |
| **19 资本结构思维** | 债务与权益平衡 | 收入结构 | 我是否过度依赖单一收入？ |
| **20 战略定位** | 在哪里竞争 | 职业方向 | 我是否在错误赛道努力？ |
| **21 竞争优势（护城河）** | 难复制能力 | 个人品牌 | 我的不可替代性是什么？ |
| **22 数据驱动决策** | 用数据验证假设 | 副业/内容 | 我是否只凭感觉决策？ |
| **23 伦理与治理** | 长期信任机制 | 合作关系 | 这是否损害长期信誉？ |
| **24 长期价值创造** | 持续价值最大化 | 人生规划 | 这是否可复利10年以上？ |
