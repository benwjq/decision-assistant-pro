| 维度 | 分析内容 |
|------|----------|
| 供应商议价能力 | 原材料供应集中度、价格敏感度 |
| 买方议价能力 | 客户集中度、价格敏感度 |

## 4. 决策分析

### 相关成本分析
```
增量成本 = 差量成本 = 边际成本
```

### 特殊订单决策
```
是否接受特殊订单？
= (特殊单价 - 单位变动成本) × 数量 > 专属成本?
```

### 自制 vs 外购
```
自制成本 = 直接材料 + 直接人工 + 变动制造费用 + 专属固定成本
外购成本 = 采购价格 + 节省的变动成本
→ 选择成本较低者
```

### 关闭 vs 继续
```
继续贡献边际 > 关闭后固定成本？→ 继续
否则 → 关闭
```

## 5. 预算管理

### 全面预算构成
```
销售预算 → 生产预算 → 直接材料/人工/制造费用预算
→ 完工产品成本预算 → 销售及管理费用预算
→ 预计利润表/资产负债表/现金流量表
```

### 预算编制方法
| 方法 | 特点 |
|------|------|
| 零基预算 | 每期从零开始 |
| 滚动预算 | 持续更新（12个月滚动） |
| 弹性预算 | 按不同业务量编制 |
| 作业预算 | 基于作业成本法 |

## 6. 差异分析

### 销售差异
```
销售量差异 = (实际销量 - 预算销量) × 预算单价 × 预算贡献边际率
销售价格差异 = (实际单价 - 预算单价) × 实际销量
```

### 成本差异
```
直接材料差异 = (实际用量 - 标准用量) × 标准价格
直接人工效率差异 = (实际工时 - 标准工时) × 标准工资率
```

## 7. 投资评估

### NPV 净现值
```
NPV = Σ [CF_t / (1+r)^t] - 初始投资
> 0 → 接受
< 0 → 拒绝
```

### IRR 内部收益率
使 NPV = 0 的折现率
> 资本成本 → 接受

### 回收期
| 类型 | 计算 |
|------|------|
| 静态回收期 | 累计现金流回本时间 |
| 动态回收期 | 折现后累计现金流回本时间 |
