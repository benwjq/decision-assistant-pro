# 法考知识库（法律思维）

## 1. 法律思维核心

### 三段论推理
```
大前提（法律规则）
小前提（案件事实）
结论（法律后果）
```

### 事实 vs 法律问题
| 类型 | 处理方式 |
|------|----------|
| 事实问题 | 证据认定、"应然" vs "实然" |
| 法律问题 | 规则解释、适用 |

---

## 2. 民法思维

### 民事法律关系分析
```
主体 → 内容（权利义务）→ 客体（物/行为/智力成果/人身利益）
```

### 意思表示
| 要素 | 说明 |
|------|------|
| 主观要件 | 效果意思、表示意思、表示行为 |
| 客观要件 | 外部表示行为 |

### 法律行为效力
```
有效 ← 民事行为能力 + 意思表示真实 + 不违法 + 不违公序良俗
     ↓
无效/可撤销/效力待定
```

### 诚实信用原则
- 帝王条款，所有民事活动应遵循
- 禁止权利滥用

---

## 3. 刑法思维

### 犯罪构成 (四要件)
```
犯罪主体 → 犯罪主观方面 → 犯罪客体 → 犯罪客观方面
```

### 违法阻却事由
| 正当防卫 | 紧急避险 |
|-----------|-----------|
| 针对不法侵害 | 针对紧迫危险 |
| 防卫意图 | 避险意图 |
| 防卫对象 | 避险对象 |
| 限度条件 | 限度条件 |

### 责任阻却事由
- 未达刑事责任年龄
- 精神病
- 不可抗力/意外事件

---

## 4. 行政法思维

### 合法性审查
```
主体合法 → 权限合法 → 程序合法 → 内容合法
```

### 三大原则
| 原则 | 内容 |
|------|------|
| 合法行政 | 法无授权不可为 |
| 合理行政 | 比例原则、考虑相关因素 |
| 程序正当 | 公开、参与、回避 |

---

## 5. 证据思维

### 证据三性
| 属性 | 含义 | 举例 |
|------|------|------|
| 真实性 | 客观真实 | 未篡改的录音 |
| 关联性 | 与待证事实有关 | 指纹与案件 |
| 合法性 | 来源/形式合法 | 不得毒树之果 |

### 证明标准
```
民事：高度可能性
刑事：排除合理怀疑
行政：清楚且有说服力
```

## 6. 权利救济

### 救济途径
| 类型 | 适用场景 |
|------|----------|
| 协商 | 双方自愿 |
| 调解 | 第三方主持 |
| 仲裁 | 约定仲裁条款 |
| 诉讼 | 最后手段 |

### 诉讼时效
```
普通诉讼时效：3年
最长诉讼时效：20年（权利受损害之日起）
```

## 7. 合同分析

### 合同要素
```
要约 ← → 承诺
     ↓
合同成立 → 生效 → 履行 → 变更/解除 → 违约责任
```

### 违约责任
```
继续履行 → 采取补救措施 → 赔偿损失 → 违约金 → 定金
```

---

## 8. 民事纠纷决策框架

### 维权成本分析
```
显性成本：律师费、诉讼费、差旅费
隐性成本：时间精力、关系破裂、商誉损失
预期收益：胜诉金额、赔偿金额
→ 成本收益权衡
```

### 诉讼or和解？
| 因素 | 诉讼 | 和解 |
|------|------|------|
| 证据强度 | 强 → 诉讼 | 弱 → 和解 |
| 关系价值 | 高 → 和解 | 低 → 诉讼 |
| 时间成本 | 低 → 诉讼 | 高 → 和解 |
| 执行可能性 | 有财产 → 诉讼 | 无财产 → 和解 |

## 9. 法律风险识别

### 风险类型
| 类型 | 说明 |
|------|------|
| 合规风险 | 违反法律法规 |
| 合同风险 | 条款漏洞、履约风险 |
| 侵权风险 | 侵犯他人权益 |
| 诉讼风险 | 被起诉或败诉 |

### 风险评估矩阵
```
影响程度 × 发生概率 = 风险等级
    高 × 高 = 高风险（优先处理）
    低 × 低 = 低风险（接受/监控）
```

## 10. 类案检索

### 检索方法
```
检索事实相似 → 检索法律问题相似 → 归纳裁判规则
```

### 类案参考原则
- 指导性案例（有约束力）
- 典型案例（参考价值高）
- 生效裁判（个案参考）

## 11. 法律思维模型（人生应用版）

| 法律思维模型 | 法学中的含义 | 对应人生场景 | 可以直接问自己的问题 |
|-------------|--------------|--------------|----------------------|
| **1 法律三段论** | 规则＋事实→结论 | 重大决定 | 我是在用情绪还是规则判断？ |
| **2 权利义务对等** | 权利必伴随责任 | 婚姻/合伙 | 我愿承担对应义务吗？ |
| **3 合法性优先** | 先判断是否越界 | 职业选择 | 是否触碰长期底线？ |
| **4 程序正义** | 过程公平优先 | 团队/家庭决策 | 是否给对方表达机会？ |
| **5 过错责任** | 谁造成风险谁承担 | 投资/合作 | 风险是谁制造的？ |
| **6 注意义务** | 是否尽合理审慎 | 结婚/创业 | 我是否充分了解后果？ |
| **7 举证责任** | 主张者需证明 | 升职/谈判 | 我能证明自己的价值吗？ |
| **8 风险分配** | 风险应由适格者承担 | 收入结构 | 谁更适合承担不确定？ |
| **9 损失最小化** | 不得扩大损害 | 止损决策 | 是否该及时退出？ |
| **10 文义解释** | 先看事实本身 | 人际关系 | 我是否过度脑补？ |
| **11 目的解释** | 看规则背后目标 | 工作选择 | 真正目标是什么？ |
| **12 体系解释** | 放入整体结构判断 | 人生平衡 | 单点成功是否破坏整体？ |
| **13 利益衡量** | 多方利益权衡 | 城市/职业选择 | 哪个损失更可承受？ |
| **14 激励相容** | 制度决定行为 | 环境选择 | 这个环境鼓励什么行为？ |
| **15 交易成本** | 降低协商摩擦 | 婚姻/合作 | 长期沟通成本高吗？ |
| **16 信赖保护** | 稳定预期优先 | 信誉/关系 | 是否破坏长期信任？ |
| **17 权力制衡** | 权力需约束 | 职业依赖 | 是否过度依赖单一资源？ |
| **18 可预期性原则** | 稳定性重于偶然 | 人生规划 | 这条路是否可持续？ |
