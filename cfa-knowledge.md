# CFA 知识库

## 1. 行为金融学 - 心理偏差

### 常见偏差速查表
| 偏差 | 定义 | 决策表现 |
|------|------|----------|
| 损失厌恶 | 损失痛感 ≈ 2倍收益快感 | 过度规避风险 |
| 锚定效应 | 过度依赖初始信息 | 被第一印象束缚 |
| 过度自信 | 高估自己判断能力 | 频繁交易 |
| 从众效应 | 跟随他人决策 | 追涨杀跌 |
| 禀赋效应 | 高估已拥有物品价值 | 不愿卖出/交换 |
| 沉没成本谬误 | 因已投入而不放弃 | 继续错误决策 |
| 近因效应 | 过度重视近期事件 | 忽视长期趋势 |
| 框架效应 | 受信息呈现方式影响 | 同一问题不同选择 |
| 心理账户 | 钱被分类不同对待 | 非理性预算分配 |
| 可得性偏差 | 高估易回忆事件概率 | 过度担心飞机失事 |

### 损失厌恶公式
```
U = Σ (ΔW) / (1 + λ)  其中 λ ≈ 2.0-2.5
```
**λ 为损失厌恶系数，损失痛感约为同等收益快感的2倍**

## 2. 投资组合理论

### 现代投资组合理论 (MPT)
- **有效前沿**: 给定风险下收益最高/给定收益下风险最低的组合
- **分散化**: 相关系数 < 1 时有效降低风险
- **夏普比率**: (Rp - Rf) / σp

### 资本资产定价模型 (CAPM)
```
E(Ri) = Rf + βi × (E(Rm) - Rf)
```

## 3. 估值模型

### DCF 折现现金流
```
企业价值 = Σ [FCF_t / (1 + WACC)^t]
```

### 相对估值
- P/E, P/B, P/S, EV/EBITDA
- 股息折现模型 (DDM)

## 4. 风险度量

| 指标 | 含义 |
|------|------|
| VaR (风险价值) | 给定置信水平下的最大损失 |
| 标准差 | 波动率，越高风险越大 |
| β 系数 | 相对于市场的 最大回撤 | 从峰值到谷底的最大跌幅 |

## 5.系统性风险 |
| 投资决策框架

### 七步决策流程
1. 定义投资目标
2. 确定约束条件（时间、流动性、税收）
3. 形成投资观点
4. 资产配置
5. 选择投资工具
6. 执行与监控
7. 绩效归因

### 风险偏好评估
```
保守型 → 债券、货币基金
平衡型 → 债券+股票混合
进取型 → 股票、另类投资
```

## 6. 行为纠偏策略

| 偏差 | 纠偏方法 |
|------|----------|
| 损失厌恶 | 事前制定规则，触发时严格执行 |
| 过度自信 | 记录决策日志，定期复盘 |
| 禀赋效应 | 问"如果没拥有，还会买吗？" |
| 沉没成本 | 只看未来成本与收益 |
| 从众 | 独立思考，问"逻辑是什么？" |

## 7. CFA 投资思维模型（人生应用版）

| CFA 投资思维模型 | 金融中的含义 | 对应人生场景 | 可以直接问自己的问题 |
|------------------|--------------|--------------|----------------------|
| **1 期望收益（Expected Value）** | 概率 × 收益 | 创业 / 转行 | 即使成功率低，一旦成功是否巨大？ |
| **2 概率分布思维** | 看结果区间而非单点 | 内容创作 / 新行业 | 最好的结果有多高上限？ |
| **3 风险≠失败（波动观）** | 波动不是坏事 | 职业探索 | 短期不稳定是否换来长期安全？ |
| **4 贝叶斯更新** | 新信息持续修正判断 | 职业路径 | 我是否允许改变方向？ |
| **5 基准率（Base Rate）** | 先看统计规律 | 创业决策 | 这个选择历史成功率如何？ |
| **6 折现现金流（DCF）** | 未来价值折现 | 读研 / 长期关系 | 未来收益是否值得现在付出？ |
| **7 内在价值 vs 市场价格** | 价格可能错 | 社会评价 | 别人认可是否等于真实价值？ |
| **8 安全边际** | 留出误差空间 | 离职 / 投资 | 即使判断错还能活吗？ |
| **9 机会成本** | 放弃的替代收益 | 留在舒适区 | 我正在拒绝什么可能？ |
| **10 资本配置** | 资源流向高回报处 | 时间管理 | 精力是否投入最高回报领域？ |
| **11 分散化（Diversification）** | 不押单一资产 | 收入结构 | 是否只有一种身份来源？ |
| **12 相关性思维** | 同跌才是真风险 | 城市+行业选择 | 我的风险是否集中？ |
| **13 下行风险控制** | 先避免毁灭 | 创业 | 最坏情况是否可承受？ |
| **14 杠杆效应** | 放大收益与风险 | 个人IP / 技术 | 哪件事能指数放大影响力？ |
| **15 流动性（Liquidity）** | 能否快速退出 | 职业技能 | 我的能力能否随时变现？ |
| **16 资产配置（Asset Allocation）** | 决定长期收益 | 行业选择 | 我站在增长赛道吗？ |
| **17 生命周期投资** | 年龄决定风险 | 人生规划 | 当前阶段是否过于保守？ |
| **18 再平衡（Rebalancing）** | 定期调整组合 | 人生复盘 | 是否3–5年重新评估方向？ |
| **19 风险预算** | 风险额度有限 | 同时做多改变 | 是否同时承担太多不确定？ |
| **20 损失厌恶** | 害怕亏损 | 不敢离开旧路径 | 我是因为害怕损失吗？ |
| **21 从众偏误** | 群体导致泡沫 | 跟风选择 | 我是真的想要还是大家都在做？ |
| **22 过度自信偏差** | 高估能力 | 顺境决策 | 成功是能力还是环境？ |
| **23 锚定效应** | 被初始信息锁定 | 第一份工资/关系 | 我是否被过去价格困住？ |
| **24 心理账户** | 错误分类资源 | 时间浪费 | 为什么对时间比钱更随意？ |
