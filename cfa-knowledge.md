# CFA 知识库

## 1. 行为金融学 - 心理偏差

### 常见偏差速查表
| 偏差 | 定义 | 决策表现 |
|------|------|----------|
| 损失厌恶 | 损失痛感 ≈ 2倍收益快感 | 过度规避风险 |
| 锚定效应 | 过度依赖初始信息 | 被第一印象束缚 |
| 过度自信 | 高估自己判断能力 | 频繁交易 |
| 从众效应 | 跟随他人决策 | 追涨杀跌 |
| 禀赋效应 | 高估已拥有物品价值 | 不愿卖出/交换 |
| 沉没成本谬误 | 因已投入而不放弃 | 继续错误决策 |
| 近因效应 | 过度重视近期事件 | 忽视长期趋势 |
| 框架效应 | 受信息呈现方式影响 | 同一问题不同选择 |
| 心理账户 | 钱被分类不同对待 | 非理性预算分配 |
| 可得性偏差 | 高估易回忆事件概率 | 过度担心飞机失事 |

### 损失厌恶公式
```
U = Σ (ΔW) / (1 + λ)  其中 λ ≈ 2.0-2.5
```
**λ 为损失厌恶系数，损失痛感约为同等收益快感的2倍**

## 2. 投资组合理论

### 现代投资组合理论 (MPT)
- **有效前沿**: 给定风险下收益最高/给定收益下风险最低的组合
- **分散化**: 相关系数 < 1 时有效降低风险
- **夏普比率**: (Rp - Rf) / σp

### 资本资产定价模型 (CAPM)
```
E(Ri) = Rf + βi × (E(Rm) - Rf)
```

## 3. 估值模型

### DCF 折现现金流
```
企业价值 = Σ [FCF_t / (1 + WACC)^t]
```

### 相对估值
- P/E, P/B, P/S, EV/EBITDA
- 股息折现模型 (DDM)

## 4. 风险度量

| 指标 | 含义 |
|------|------|
| VaR (风险价值) | 给定置信水平下的最大损失 |
| 标准差 | 波动率，越高风险越大 |
| β 系数 | 相对于市场的 最大回撤 | 从峰值到谷底的最大跌幅 |

## 5.系统性风险 |
| 投资决策框架

### 七步决策流程
1. 定义投资目标
2. 确定约束条件（时间、流动性、税收）
3. 形成投资观点
4. 资产配置
5. 选择投资工具
6. 执行与监控
7. 绩效归因

### 风险偏好评估
```
保守型 → 债券、货币基金
平衡型 → 债券+股票混合
进取型 → 股票、另类投资
```

## 6. 行为纠偏策略

| 偏差 | 纠偏方法 |
|------|----------|
| 损失厌恶 | 事前制定规则，触发时严格执行 |
| 过度自信 | 记录决策日志，定期复盘 |
| 禀赋效应 | 问"如果没拥有，还会买吗？" |
| 沉没成本 | 只看未来成本与收益 |
| 从众 | 独立思考，问"逻辑是什么？" |
